# 🚀 Full-Stack Custom Kanban Board

A professional, high-performance Kanban application built with a modern tech stack. This project focuses on clean architecture, smooth UX, and containerized deployment.

## 🛠 Tech Stack

- **Frontend:** React.js (Vite), Custom CSS (Zero libraries), HTML5 Drag & Drop API.
- **Backend:** Python (FastAPI), SQLAlchemy (ORM).
- **Database:** SQLite (Persistent storage).
- **DevOps:** Docker, Docker Compose.

## ✨ Key Features

- **Custom Drag & Drop:** Built from scratch using native browser API (no heavy libraries).
- **Persistent Storage:** All tasks are saved in a real SQL database.
- **Optimistic UI:** Super fast interface - changes are reflected instantly before server confirmation.
- **Glassmorphism Design:** Modern Dark Mode UI with smooth animations.
- **Dockerized:** Entire environment starts with a single command.
